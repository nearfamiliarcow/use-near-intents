## Primary Accent
- Orange: `#fb4d01` — used for the "Intents" wordmark, CTAs, active states, links
- Orange light (tint): `#fff0ea` — hover backgrounds, tag backgrounds
- Orange dark: `#d93f00` — reserved for pressed states if needed

## Neutral Palette (from NEAR AI Brand Guidelines)
- Black: `#000000` (PMS Black C)
- Dark Grey: `#272727` (PMS 426C) — secondary text, body copy
- Grey: `#a7a7a7` (PMS 173-3C) — muted text, nav items, captions
- Off White: `#eeeeeb` (PMS 179C) — page background
- White: `#ffffff` — card surfaces, header/footer backgrounds

## UI Tokens (as defined in app.css)
- `--color-near-bg`: `#eeeeeb`
- `--color-near-surface`: `#ffffff`
- `--color-near-text`: `#000000`
- `--color-near-text-secondary`: `#272727`
- `--color-near-text-muted` / `--color-near-gray`: `#a7a7a7`
- `--color-near-border`: `#d8d8d4`
- `--color-near-orange`: `#fb4d01`
- `--color-near-orange-light`: `#fff0ea`

## Legacy Aliases (backward compat — all map to orange)
- `near-green` → `#fb4d01`
- `near-green-dark` → `#fb4d01`
- `near-green-light` → `#fff0ea`

## Status Colors (unchanged)
- Live: `#059669`
- Building: `#d97706`
- Opportunity: `#2563eb`

## What NOT to use
- Do NOT use the NEAR AI blue palette (`#0091FD`, `#83DCFF`) — that's for NEAR AI, not NEAR Intents
- Do NOT use green (`#00c853`, `#00963f`) — original site used these but they are off-brand
